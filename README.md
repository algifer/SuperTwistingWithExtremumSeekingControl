To visualize the results shown in **Figures 3, 4, and 5** presented in the article, go to the folders DOE_Cumulated and DOE_NonCumulated.  These contain the data generated for the accumulated and non-accumulated point cases.  The data for Figure 3 can be found in the DOE_NonCumulated folder.  * To reproduce the results of Figure 3 (the particular case), go to the DOE_NonCumulated folder and run the file:    `main_generate_response_surface_with_particular_case_points.m`  * To reproduce the results of Figure 4 (the non-accumulated case), go to the DOE_NonCumulated folder and run the file:    `main_generate_response_surface_with_cumulated_points.m`    If you want to see how the experimental design data are generated, run the file:    `main_generate_noncumulated_points.m`  * To reproduce the results of Figure 5 (the accumulated case), go to the DOE_Cumulated folder and run the file:    `main_generate_response_surface_with_cumulated_points.m`    If you want to see how the experimental design data are generated, run the file:    `main_generate_cumulated_points.m`NOTE: To generate the response surface models, it is necessary to upload the Excel files generated for each set of accumulated and non-accumulated points into the MATLAB toolbox called Regression Learner, and select the no validation option.This is because we are **not interested in validating the model using standard machine learning methods, but only in obtaining a response surface with specific characteristics described in the article (in the Remarks section), which we use to tune the ESC-ST controller gains.2 - To reproduce the results of Figures 6, 7, and 8 presented in the article, run the file:  `MainESC_ST_Differentiator_Mechanistic_And_2_Interpolated_model.m`3 - To reproduce the results of Figures 9, 10, 11, and 12 presented in the article, run the file:  `MainESC_ST_Differentiator_with_real_model.m`---28/01/2026 – Second revision1.
